## Adding the Transcription Card Template to Anki

### 1. Create a New Deck

Open Anki and create a new deck for your transcription cards.

### 2. Create the Note Type

1. Open the deck and click **Add** to open the Add Card dialog.
2. Click the **Type** field at the top of the window, then click **Manage**.
3. Click **Add**, choose **Basic** as the base type, and click **OK**.
4. Name the new note type `Transcription`, then close the Note Type window.

### 3. Configure the Fields

1. Back in the Add Card dialog, click the **Type** field and select **Transcription**.
2. Open the card template editor by clicking the **Cards…** button.
3. Update the note type to use two fields: `Audio` and `Answer`.

### 4. Apply the Card Templates

In the card template editor, paste the provided content into each section:

- **Front Template** — controls what is shown on the front of the card.
- **Back Template** — controls what is revealed when you flip the card.
- **Styling** — controls the appearance of both sides.
